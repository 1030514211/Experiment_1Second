# Experiment_1Second
轮转法（processing)</br>
左上角为进程名字，旁边为到达时间，下面为所需时间。</br>
先点击sort，再点击RR，然后就可以开始键盘操纵。</br>
此时按一下'A'或'a'，则所需要的运行时间减一，在鼠标点击一次sort，重新排序。</br>
重复此过程（按键，鼠标，按键，鼠标），直至运行时间为0。</br>
